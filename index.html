GI Health App

A simple and practical way to track digestive health and daily habits.

Motivation

This project was created to provide a straightforward way for users to track their gastrointestinal health. Many existing tools are either too complex or not tailored to digestive health, so the goal was to design something simple, organized, and easy to use on a daily basis.

Installation

  1. Download the APK file from the link below
  2. Transfer the file to your Android device
  3. Enable "Install unknown apps" in your device settings
  4. Open the APK file and follow the installation steps
